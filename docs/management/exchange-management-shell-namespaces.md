# <a name="exchange-management-shell-namespaces"></a>Espaces de noms de l’environnement de ligne de commande Exchange Management Shell

Recherchez des informations sur les espaces de noms pour Exchange Management Shell dans Exchange.
  
**S’applique à :** Exchange Online | Exchange Server 2013 | Office 365
  
Exchange Les applications Management Shell pour Exchange Online, Exchange Online dans le cadre de Office 365 ou une version de Exchange commençant par Exchange 2013 interagissent avec l’environnement Windows PowerShell via les types exposés par le **Espace de noms System.Management.Automation.** Les cmdlets Exchange Management Shell utilisent des types provenant d’Exchange bibliothèques. Cette section fournit des informations de référence sur les types et les interfaces des bibliothèques Exchange utilisées par les cmdlets Exchange Management Shell.
