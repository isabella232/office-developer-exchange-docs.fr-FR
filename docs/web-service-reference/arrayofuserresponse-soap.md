# <a name="arrayofuserresponse-soap"></a>ArrayOfUserResponse (SOAP)

**L’élément ArrayOfUserResponse** contient un tableau d’éléments [UserResponse (SOAP).](userresponse-soap.md) 
  
```XML

   <UserResponse/>
</ArrayOfUserResponse>
```
