# <a name="authenticationmethods-soap"></a>AuthenticationMethods (SOAP)

L’élément **AuthenticationMethods** décrit les méthodes d’authentification disponibles pour un client Web. 
  
```XML
<AuthenticationMethods/>
```

 **chaîne**
## <a name="attributes-and-elements"></a>Attributs et éléments

Les sections suivantes décrivent les attributs, les éléments enfants et les éléments parents.
  
### <a name="attributes"></a>Attributs

Aucune.
  
### <a name="child-elements"></a>Éléments enfants

Aucun.
  
### <a name="parent-elements"></a>Éléments parents

|**Élément**|**Description**|
|:-----|:-----|
|[WebClientUrl (SOAP)](webclienturl-soap.md) <br/> |Représente l’URL du client Outlook Web App.  <br/> |
   
## <a name="text-value"></a>Valeur texte

La valeur de texte de l’élément **AuthenticationMethods** est l’URL du client Outlook Web App.
