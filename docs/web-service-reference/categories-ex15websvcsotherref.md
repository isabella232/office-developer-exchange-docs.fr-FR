# <a name="categories"></a>Catégories

L’élément de **catégories** contient une collection de chaînes qui identifient les catégories auquel appartient un élément dans la boîte aux lettres. 
  
```XML
<Categories>
   <String/>
</Categories>
```

 **ArrayOfStringsType**
## <a name="attributes-and-elements"></a>Attributs et éléments

Les sections suivantes décrivent les attributs, éléments enfants et éléments parents.
  
### <a name="attributes"></a>Attributs

Aucun.
  
### <a name="child-elements"></a>Éléments enfants

|**Élément**|**Description**|
|:-----|:-----|
|[String](string.md) <br/> |Contient une chaîne qui identifie une seule catégorie.  <br/> |

|**Élément**|**Description**|
|:-----|:-----|
