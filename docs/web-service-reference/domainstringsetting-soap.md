# <a name="domainstringsetting-soap"></a>DomainStringSetting (SOAP)

**L’élément DomainStringSetting représente** un paramètre de domaine dont la valeur est de type chaîne. 
  
```XML
<DomainStringSetting>
   <Name/>
   <Value/>
</DomainStringSetting>
```

 **DomainStringSetting**
## <a name="attributes-and-elements"></a>Attributs et éléments

Les sections suivantes décrivent les attributs, les éléments enfants et les éléments parents.
  
### <a name="attributes"></a>Attributs

Aucune.
  
### <a name="child-elements"></a>Éléments enfants

|**Élément**|**Description**|
|:-----|:-----|
|[Name (SOAP)](name-soap.md) <br/> |Représente le nom d’un paramètre.  <br/> |
|[Value (SOAP)](value-soap.md) <br/> |Représente une valeur de paramètre utilisateur.  <br/> |
   
### <a name="parent-elements"></a>Éléments parents

Aucun.
  
## <a name="text-value"></a>Valeur de texte

Aucune.
