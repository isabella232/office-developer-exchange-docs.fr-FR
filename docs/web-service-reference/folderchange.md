# <a name="folderchange"></a>FolderChange

**L’élément FolderChange** représente une collection de modifications à effectuer sur un seul dossier. 
  
- [UpdateFolder](updatefolder.md) 
- [FolderChanges](folderchanges.md) 
- [FolderChange](folderchange.md)
  
```xml
<FolderChange>
   <FolderId/>
   <Updates/>
</FolderChange>
```

```xml
<FolderChange>
   <DistinguishedFolderId/>
   <Updates/>
</FolderChange>
```

**FolderChangeType**

## <a name="attributes-and-elements"></a>Attributs et éléments

Les sections suivantes décrivent les attributs, les éléments enfants et les éléments parents.
  
### <a name="attributes"></a>Attributs

Aucune.
  
### <a name="child-elements"></a>Éléments enfants

|**Élément**|**Description**|
|:-----|:-----|
|[FolderId](folderid.md) <br/> |Contient l’identificateur et la clé de modification d’un dossier à mettre à jour.  <br/> |
|[DistinguishedFolderId](distinguishedfolderid.md) <br/> |Identifie les dossiers MicrosoftExchange Server 2007 qui peuvent être référencés par leur nom.  <br/> |
|[Updates (Folder)](updates-folder.md) <br/> |Définit le type de mise à jour effectuée sur un dossier identifié par l’élément [FolderId](folderid.md) ou [DistinguishedFolderId.](distinguishedfolderid.md)  <br/> |
   
### <a name="parent-elements"></a>Éléments parents

|**Élément**|**Description**|
|:-----|:-----|
|[FolderChanges](folderchanges.md) <br/> |Représente une collection de modifications pour un dossier.  <br/> Voici l’expression XPath de cet élément :  <br/>  `/UpdateFolder/FolderChanges` <br/> |
   
## <a name="remarks"></a>Remarques

Le schéma qui décrit cet élément se trouve dans le répertoire virtuel EWS de l’ordinateur exécutant Exchange Server 2007 sur qui le rôle serveur d’accès au client est installé.
