# <a name="getfederationinformationrequestmessage-soap"></a>GetFederationInformationRequestMessage (SOAP)

**L’élément GetFederationInformationRequestMessage** prépare un appel au serveur pour demander des données de configuration pour le service d’jeton de sécurité (STS). 
  
```XML
<GetFederationInformationRequestMessage>
   <Request/>
</GetFederationInformationRequestMessage>

```

Les sections suivantes décrivent les attributs, les éléments enfants et les éléments parents.
