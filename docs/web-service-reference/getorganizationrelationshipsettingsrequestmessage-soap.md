# <a name="getorganizationrelationshipsettingsrequestmessage-soap"></a>GetOrganizationRelationshipSettingsRequestMessage (SOAP)

**L’élément GetOrganizationRelationshipSettingRequestMessage** représente une demande d’opération [SOAP (GetOrganizationRelationshipSettings).](getorganizationrelationshipsettings-operation-soap.md) **L’élément GetOrganizationRelationshipSettingRequestMessage** est uniquement à usage interne. Cet élément n’est pas utilisé par les clients. 
  
```XML
<GetOrganizationRelationshipSettingRequestMessage>
   <Request/>
</GetOrganizationRelationshipSettingRequestMessage>
```
