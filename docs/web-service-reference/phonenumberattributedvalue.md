# <a name="phonenumberattributedvalue"></a>PhoneNumberAttributedValue

**L’élément PhoneNumberAttributedValue** spécifie une instance d’un tableau de numéros de téléphone et leurs attributions associées. 
  
```XML
<PhoneNumberAttributedValue>
   <Value/>
   <Attributions/>
</PhoneNumberAttributedValue>
```

 **PhoneNumberAttributedValueType**
## <a name="attributes-and-elements"></a>Attributs et éléments

Les sections suivantes décrivent les attributs, les éléments enfants et les éléments parents.
  
### <a name="attributes"></a>Attributs

Aucune.
