# <a name="port-soap"></a>Port (SOAP)

**L’élément Port** représente le numéro de port à utiliser pour le protocole. 
  
```XML
<Port/>
```

 **int**
## <a name="attributes-and-elements"></a>Attributs et éléments

Les sections suivantes décrivent les attributs, les éléments enfants et les éléments parents.
  
### <a name="attributes"></a>Attributs

Aucune.
  
### <a name="child-elements"></a>Éléments enfants

Aucun.
  
### <a name="parent-elements"></a>Éléments parents

|**Élément**|**Description**|
|:-----|:-----|
|[ProtocolConnection (SOAP)](protocolconnection-soap.md) <br/> |Représente la connexion de protocole du client Web serveur.  <br/> |
   
## <a name="text-value"></a>Valeur texte

La valeur est le numéro de port à utiliser pour le protocole.
