# <a name="protocolconnections-soap"></a>ProtocolConnections (SOAP)

L’élément **ProtocolConnections** contient zéro ou plusieurs connexions de protocole. 
  
```XML
<ProtocolConnections>
   <ProtocolConnection/>
</ProtocolConnections>
```
