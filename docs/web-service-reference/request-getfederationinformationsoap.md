# <a name="request-getfederationinformation-soap"></a>Request (GetFederationInformation) (SOAP)

**L’élément Request** représente une [demande GetFederationInformationRequest (SOAP).](getfederationinformationrequest-soap.md) 
  
```XML
<Request>
   <Domain/>
</Request>
```

 **GetFederationInformationRequest**
## <a name="attributes-and-elements"></a>Attributs et éléments

Les sections suivantes décrivent les attributs, les éléments enfants et les éléments parents.
  
### <a name="attributes"></a>Attributs

Aucune.
  
### <a name="child-elements"></a>Éléments enfants

|**Élément**|**Description**|
|:-----|:-----|
|[Domaine (GetFederationInformation) (SOAP)](domain-getfederationinformationsoap.md) <br/> |Identifie le domaine qui a une confiance de fédération.  <br/> |
   
### <a name="parent-elements"></a>Éléments parents

|**Élément**|**Description**|
|:-----|:-----|
|[GetFederationInformationRequestMessage (SOAP)](getfederationinformationrequestmessage-soap.md) <br/> |Prépare un appel au serveur pour demander des données de configuration pour le service d’jeton de sécurité (STS).  <br/> |
