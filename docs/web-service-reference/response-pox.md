# <a name="response-pox"></a>Response (POX)


  
**L’élément Response** contient la réponse du service de découverte automatique. 
  
[AutoDiscover (POX)](autodiscover-pox.md)
  
[Response (POX)](response-pox.md)
  
```xml
<Response>
   <User/>
   <Account/>
</Response>
```

## <a name="attributes-and-elements"></a>Attributs et éléments

Les sections suivantes décrivent les attributs, les éléments enfants et les éléments parents.
  
### <a name="attributes"></a>Attributs

Aucune.
  
### <a name="child-elements"></a>Éléments enfants

|**Élément**|**Description**|
|:-----|:-----|
|[User (POX)](user-pox.md) <br/> |Fournit des informations spécifiques à l’utilisateur. Cet élément est facultatif.  <br/> |
|[Account (POX)](account-pox.md) <br/> |Spécifie les paramètres de compte de l’utilisateur.  <br/> |
   
### <a name="parent-elements"></a>Éléments parents

|**Élément**|**Description**|
|:-----|:-----|
|[AutoDiscover (POX)](autodiscover-pox.md) <br/> |Élément racine dans une réponse de découverte automatique.  <br/> |
