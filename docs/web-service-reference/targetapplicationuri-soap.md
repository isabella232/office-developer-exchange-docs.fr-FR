# <a name="targetapplicationuri-soap"></a>TargetApplicationUri (SOAP)

**L’élément TargetApplicationUri** définit l’URI de l’application cible. **L’élément TargetApplicationUri** est à usage interne uniquement. Cet élément n’est pas utilisé par les clients. 
  
```XML
<TargetApplicationUri/>
```

 **anyURI**
## <a name="attributes-and-elements"></a>Attributs et éléments

Les sections suivantes décrivent les attributs, les éléments enfants et les éléments parents.
  
### <a name="attributes"></a>Attributs

Aucune.
  
### <a name="child-elements"></a>Éléments enfants

Aucun.
  
### <a name="parent-elements"></a>Éléments parents

|**Élément**|**Description**|
|:-----|:-----|
|[OrganizationRelationshipSettings (SOAP)](organizationrelationshipsettings-soap.md) <br/> |Représente une liste de relations organisationnelles pour une organisation unique  <br/> |
   
## <a name="remarks"></a>Remarques

Cet élément définit l’URI cible de l’organisation externe.

|:-----|:-----|
|Espace de noms  <br/> |https://schemas.microsoft.com/exchange/2010/Autodiscover  <br/> |
